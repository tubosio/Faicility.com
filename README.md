# Faicility.com Website

Statische website voor Faicility, klaar om te publiceren via GitHub Desktop + GitHub Pages.

## Publiceren met GitHub Desktop

1. Open GitHub Desktop.
2. Kies **File > Add Local Repository...** en selecteer:
   `/Users/niekbuis/Documents/Faicility.com`
3. Als dit nog geen repository is, kies **Create a Repository**.
4. Commit alle wijzigingen.
5. Klik **Publish repository**.
6. Zet in GitHub (web):
   - **Settings > Pages**
   - **Build and deployment**: `Deploy from a branch`
   - **Branch**: `main` en map `/ (root)`

## Belangrijk

- Alle lokale asset-paden zijn relatief gemaakt (geschikt voor GitHub Pages).
- Startpagina: `index.html`
- Styles: `resources/css/style.css`
- Scripts: `resources/js/script.js`
